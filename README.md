## Tech Stack

**Frontend**
- React 19 + TypeScript + Vite
- Tailwind CSS 4
- Firebase (Auth, Firestore)
- React Router DOM 7

**Backend**
- FastAPI + PostgreSQL
- JWT + Google OAuth2
- WebSocket (real-time tracking)
- Redis + Docker

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
docker-compose up
```
